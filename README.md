# Testimonial Cards Component 
A Clean, responsive, and reusable Testimonial Cards UI component built with modern front-end best practices. Designed for easy drop-in use across landing pages, portfolios, product sites, and brand experiences. 

## Features
- Responsive card layout using Flexbox/Grid
- Accessible semantic HTML structure
- Smooth hover interactions and subtle motion
- Modular, scalable CSS (SASS-friendly)
- Works with any design system or color palette


## 🚀 Usage
1. Copy the component folder into your project.
2. Link the stylesheet in your HTML:
   ```html
   <link rel="stylesheet" href="styles.css">
3. Paste the testimonial card markup where you want it to appear.
4. Customize colors, spacing, and typography to match your brand.

   Customization
   -Update avatar images in /assets
   Adjust theme variables (colors, shadows, radii)
   Optional JS hook for sliders, carousels, or animations
